# memohub

### Place where you can store your memes.


### Commands:
- `make up` - Start project
- `make stop` - Stop project
- `make cs` - Create superuser
- ... *list of all available commands you can see in `Makefile`*

### Built with:
- `Django framework`
- `PostgreSQL`
